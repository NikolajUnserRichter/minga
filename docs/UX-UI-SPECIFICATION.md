## 6. Responsive Design

### 6.1 Breakpoints

| Breakpoint | Breite | Verhalten |
|------------|--------|-----------|
| Mobile | < 640px | Sidebar collapsed, vereinfachte Tabellen |
| Tablet | 640px - 1024px | Sidebar toggle, kompakte Karten |
| Desktop | > 1024px | Volle Sidebar, alle Features |

### 6.2 Mobile-Anpassungen

- **Dashboard**: KPI-Karten stapeln vertikal
- **Tabellen**: Horizontales Scrollen oder Card-Ansicht
- **Sidebar**: Als Overlay/Drawer
- **Formulare**: Vollbreite Inputs
- **Charts**: Vereinfacht oder als Sparklines
